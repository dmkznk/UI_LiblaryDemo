Connect CSS

Put the stylesheet <link> into your <head> before all other stylesheets to load this library CSS

Link example: 

    <link rel="stylesheet" href="yourFolder/lib.css"/>

Connect JS

Place the JS <script> near the end of your page, right before the closing </body> tag.

Link example: 

    <script src="yourFolder/lib.js"></script>

Hendlers


    on(eventName, callback); 

    // As firs argument this method takes 'eventName',
    // such as 'click', 'submit', 'mouseover', 'keydown'...
    // And the second argument is 'callback' function, what you want to do on this event.

    off(eventName, callback); 

    // This method is the opposite to previous one. It removes event,
    // that you don't need anymore.

    click(handler); 

    // As you can guess, this method takes a function that you want to call on 'click'.

Effects


    fadeIn(duration, display, finish) 

    // This method takes 'duration' of the animation, and 'display' type, that you want 
    // your element to have, but it is optional, as default it would be 'block'.
    // Also it takes optionat argument 'finish', this should be a function that you want you run,
    // after animation ends.

    fadeOut(duration, finish)

    // This one is the opposite to previous one. It dont need 'display' settings because it always 
    // becomes 'none', after 'duration' ends. 'Finish' here works apsolutely the same,
    // as in previous method.

    fadeToggle(duration, display, finish) 

    // As you can guess, this one is kind of two in one. If your element has 'display: none;',
    // this method will set 'display' to what you put as argument, or to 'block',
    // if you haven't put any.
    // And if your element is visible, it would be set to 'display: none;',
    // after duration ends. 
    // 'Finish' here works apsolutely the same, as in two previous methods.

Display


    hide();

    // This method dont takes any arguments, but you can use it in a chaining way.
    // The aim of it is to hide your element, other word to set 'display' to 'none'. 
    
    // For example, if you run it like this, the body page would disappeare:
    $('body').hide();

    show();

    // Opposite to previous one. This method removes 'none' of 'display' parameter,
    // and set it to your default settings.

    toggle();

    // It's like two in one. If element visible, this methot would hide it,
    // if opposite, your element would become visible.

Actions


   html(content);

   // This method takes 'content', that you want to put into your
   // element or elements. For example some HTML elements, or just text.

   eq(index);

   // This one takes 'index' of element, and returs you an object with this element.

   index();

   // Returns you index of element on which it was caused.

   find(selector);

   // Takes 'selector' of element that you want to find, and returs an object
   // width this element or elements.

   siblings(); 

   // Returns an object with all sibling elements of the element on which it was caused.

Classes


   addClass(classNames);

   // As you probably guessed, this method add classes or one class to element
   // on which it was caused.

   removeClass(classNames);

   // The same as previous, but opposite.

   toggleClass(className);

   // This method takes only one class name, that you want to toggle.

Atributes


   addAttribute(name, value);

   // This one add attribte to an element. 'Name' is a name of attribute,
   // 'value' is a value. Second argument is optional, as default it set in 'true'. 

  deleteAttribute(name);
  
  // Deletes atribute from an element. Takes 'name' of attribute that you want,
  // to be deteted. 

  toggleAttribute(name);

  // This method takes atribute 'name', that you want to toggle.

Requests


  get(url, dataTypeAnswer);  
  
  // First argument is a link to server, from where you want to get data.
  // Second argument is a type of data. Last argument is optional, as default it
  // set to JSON. You can put 'text' or 'blob', as second argument, if you want 
  // answer in such format. 

  // This method returns promise, so you need to use JS native method 'then',
  // to see desired result.

  // For example, if you run such code:
  $('body').get('https://jsonplaceholder.typicode.com/posts/1')
  .then(json => console.log(json)); 

  // In console you will see the following:

  {
    id: 1,
    title: '[...]',
    body: '[...]',
    userId: 1
  }

  post(url, data, dataTypeAnswer)

  // This method is for creating post on server. First argument is a link 
  // to server, second one, is a data that you want to post, thrid one, format of
  // answer. Last one is optional, as default it set to 'text'. It can also be,
  // 'json' or 'blob'.

  // Same as previous, this method returns promise.
